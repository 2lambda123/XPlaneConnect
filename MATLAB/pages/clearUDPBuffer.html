clearUDPBuffer

Script that clears an UDP Socket Buffer

Inputs

  • Socket: UDP Socket to be cleared

Outputs

  • Socket: UDP Socket

Use

1. import XPlaneConnect.*;
2. Socket = openUDP(49005);
3. Socket = clearUDPBuffer(Socket);
